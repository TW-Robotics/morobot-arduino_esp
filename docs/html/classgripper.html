[in]degOpenLimitGripper will not open above this limit (would harm hardware)
[in]gearRatioThe gearRatio defines how many mm (or degrees) the gripper moves per degree motor movement
[in]closingWidthOffsetOpening width of the gripper when it is completely closed (for parallel grippers only
