Uses given coordinates to calculate the motor angles to reach this position (Solve inverse kinematics). This function does only calculate the angles of the motors and stores them internally. Use moveToPose(x,y,z) to actually move the robot.

Parameters

[in]yDesired x-position of TCP
[in]zDesired x-position of TCP
Returns
Returns true if the position is reachable; false if it is not.
